getblockcount

Returns the number of blocks in the longest blockchain.

Result:

(numeric) The current block count

Examples:

particl-cli getblockcount curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getblockcount", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:51935/
